# firebase-eval

## Un formulario de ingreso de usuarios

## Vue.js, VueX y Firebase con Firestore
## Para instalar las dependencias y correr la aplicación deberá correr los siguientes comandos en la terminal:
## 1 npm install
## 2 npm run dev
